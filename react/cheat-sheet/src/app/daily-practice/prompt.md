## Post Filtering


### Step 1
Create a React app that fetches posts from an endpoint and displays it


### Step 2
Enable filtering on the posts, using a text field, either filter by body, text, or id

The list of posts should automatically rerender to show the corresponding posts

### Step 3
enable sorting the posts
Should be able to sort by 
    - Body ascending/descending
    - Title ascending/descending
    - id/user id ascending/descending

### Step 3 1/2
Enable pagination for data

### Step 4
Using flex, display the posts as rows and columns of posts, that adjusts how many posts there are in a row based on width
